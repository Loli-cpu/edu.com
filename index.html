    Basic Java Code Examples with Descriptions

    Java is a high-level, object-oriented programming language that is compiled into bytecode and run on the Java Virtual Machine (JVM). Below are some basic Java code examples with comments explaining how they work.

    1. HelloWorld Program

    public class HelloWorld {
    // The main method is the entry point of any Java application
    public static void main(String[] args) {
        // This line prints "Hello, World!" to the console
        System.out.println("Hello, World!");
    }
}

    2. Variables and Data Types

    public class VariablesExample {
    public static void main(String[] args) {
        // Declare an integer variable and initialize it
        int age = 25;
        // Declare a double variable for floating-point numbers
        double salary = 55000.50;
        // Declare a String variable to hold text
        String name = "Alice";
        
        // Print the values of the variables to the console
        System.out.println("Name: " + name);
        System.out.println("Age: " + age);
        System.out.println("Salary: " + salary);
    }
}

    3. Control Flow (If-Else Statement)

    public class ControlFlowExample {
    public static void main(String[] args) {
        // Declare an integer variable
        int number = 10;
        
        // Check if the number is greater than 0
        if (number > 0) {
            // This block executes if the condition is true
            System.out.println("The number is positive.");
        } else if (number < 0) {
            // This block executes if the first condition is false and this condition is true
            System.out.println("The number is negative.");
        } else {
            // This block executes if none of the above conditions are true
            System.out.println("The number is zero.");
        }
    }
}

    4. Loops (For Loop)

    public class LoopExample {
    public static void main(String[] args) {
        // For loop that iterates from 0 to 4
        for (int i = 0; i < 5; i++) {
            // Print the current iteration number
            System.out.println("Iteration: " + i);
        }
    }
}

    5. Functions (Methods)

    public class MethodExample {
    public static void main(String[] args) {
        // Call the addNumbers method and store the result
        int result = addNumbers(5, 7);
        // Print the result of the addition
        System.out.println("Sum: " + result);
    }
    
    // Method that adds two numbers and returns the result
    public static int addNumbers(int a, int b) {
        return a + b; // Return the sum of a and b
    }
}

}
